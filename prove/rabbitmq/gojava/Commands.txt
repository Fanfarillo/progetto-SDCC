1) INSTALL SERVER ON https://www.rabbitmq.com/download.html (in rabbitmq directory)
bash InstallServer.sh

2) MAKE SURE THAT GOPATH VARIABLE CONTAINS:
/${HOME}/go
sudo nano /etc/environment
source /etc/environment

3) MAKE SURE THAT JAVA_HOME VARIABILE CONTAINS:
/usr/lib/jvm/{JAVA-JDK}
sudo nano /etc/environment
source /etc/environment

4) GENERATE go.mod AND go.sum FILES (in rabbitmq/gojava directory)
go mod init gojava
go mod tidy

5) COMPILE JAVA FILE (in rabbitmq/gojava/java directory)
mvn clean verify

6) EXECUTE CONSUMER (in rabbitmq/gojava directory)
go run ./go/Consumer.go

7) EXECUTE PRODUCER (in rabbitmq/gojava/java directory)
mvn exec:java -Dexec.mainClass="Producer"
