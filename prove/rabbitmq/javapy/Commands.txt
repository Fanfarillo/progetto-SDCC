1) INSTALL SERVER ON https://www.rabbitmq.com/download.html (in rabbitmq directory)
bash InstallServer.sh

2) INSTALL PIKA
python3 -m pip install pika --upgrade

3) MAKE SURE THAT JAVA_HOME VARIABILE CONTAINS:
/usr/lib/jvm/{JAVA-JDK}
sudo nano /etc/environment
source /etc/environment

4) COMPILE JAVA FILE (in rabbitmq/javapy/java directory)
mvn clean verify

5) EXECUTE PRODUCER (in rabbitmq/javapy directory)
python3 ./python/Producer.py

6) EXECUTE CONSUMER (in rabbitmq/javapy/java directory)
mvn exec:java -Dexec.mainClass="Consumer"
