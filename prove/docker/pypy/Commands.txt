1) INSTALL GRPCIO, DOCKER AND DOCKER-COMPOSE
python3 -m pip install grpcio
python3 -m pip install grpcio-tools
Per l'installazione di Docker, vedere la pagina ufficiale.
sudo apt-get install docker-compose-plugin

2) COMPILE PROTO FILE (in docker/pypy directory)
python3 -m grpc_tools.protoc -I. --python_out=./client --grpc_python_out=./client ./proto/ClientToServer.proto
python3 -m grpc_tools.protoc -I. --python_out=./server --grpc_python_out=./server ./proto/ClientToServer.proto

3) BUILD IMAGES AND EXECUTE CONTAINERS (in docker/pypy directory)
docker compose up --build
